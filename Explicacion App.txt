La app trata sobre el pokemon eevee y sus evoluciones.

SplashActivity: Se muestra una pantalla por un segundo aprox para una bienvenida a la app, se creo un estilo con el diseño del pokemon eevee con una imagen que esta en la carpeta drawable. Esta actividad tiene un intent que nos llevara a la LoginActivity una vez que se termine esta actividad y se le agrega un finish para no volver a esta misma.

LoginActivity: Lo primero que hice fue crear la pantalla de login para que el usuario inicise sesion, le agregue un TextView para escribir el mensaje de binvenida, luego agregue dos EditTtext,uno para colocar el usuario y otro para la contraseña asi el usuario pueda iniciar sesion, despues agregue dos ImageView para colocar las imagenes. Para que el usuario inicie sesion cree un usuario generico, para eso cree tres variables una llamada user(eevee), otra clave(pokemon123) y otra genreMale(masculina). Cree una array con tres strings llamadas "masculino", "femenino" y "no definido", que lo pondremos en un spinner para que el usuario eliga el genero, despues cree una funcion para validar esa identidad, gracias a un if si lo datos estan correctos salta un mensaje "Inicio De Sesion Exitoso" significando que los datos ingresados del usuario son correctos, gracias a un else if si uno de los campos o los dos campos estan en blanco saltara un mensaje "Complete el espacio en blanco" y con un else si el usuario o la clave o ambos son incorrectos saltara un mensaje "usuario invalido".

MainActivity: En este MainActivity sera la actividad principal donde contendremos un NavegationDrawer sera un panel donde tendra las distintas opciones que al clickear "informacion de pokemons" tendremos un fragmento llamado MainFragment que mandara al usuario donde esta la informacion de eevee y sus evoluciones, en esta opcion tendremos el recycleView donde este tendra un adapatador que tendra los datos y la vista que enlazaremos al recyleView en la cual se mostrara una lista scrollable, en este caso agregue las imagenes en drawable y los strings en el archivo strings ademas cree un paquete llamado model que tendra una dataclass que se llama pokemonModels con dos variables tipo string para almacenar la informacion, al clickear "pokedex" nos llevara a una web relacionada a esta app, al clcikear "terminos y condiciones" nos llevara a un fragmento llamado TermsConditionFragment que contendra un TextView con los terminos y condiciones de la app.
